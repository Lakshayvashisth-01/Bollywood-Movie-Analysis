# 🎬 Movie Sales Analysis Dashboard – Power BI Project

## 📌 Overview

The **Movie Sales Analysis Dashboard** is an interactive Power BI project designed to analyze movie industry performance using key business and entertainment metrics. The dashboard provides valuable insights into movie revenue, budget, ratings, and genre-wise performance through dynamic visualizations and KPI cards.

This project helps users understand trends in movie sales and evaluate the financial and audience success of films using data-driven analytics.

---

## 🚀 Features

* 📊 Interactive Power BI dashboard
* 💰 Revenue and Budget analysis
* ⭐ Movie rating insights
* 🎭 Genre-wise performance comparison
* 📈 Sales trend visualization
* 🔍 Dynamic filters and slicers
* 📋 KPI cards for quick insights

---

## 🛠️ Tools & Technologies Used

* **Power BI**
* **Data Visualization**
* **Business Intelligence**
* **Data Analysis**
* **DAX (Data Analysis Expressions)**

## 📂 Dashboard Insights

The dashboard includes:

* Total Revenue Analysis

* Top Performing Movies
* Sales Trends and Charts
